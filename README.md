## Set up

# Clone this project:
$ https://github.com/sabrinamianne/Etsy-Clone.git

# Go into the directory
$ cd Etsy-clone

# Install dependecies
$ npm install

# Create a Firebase account

# Install Firebase using npm and deploy
$ npm install -g firebase-tools
$ firebase login
$ firebase open hosting:site
